# Real-Time-Systems-Final-Project

This is a project I did for my real time systems class.

This project's main objective was to see if an Arduino UNO R3 can be used with 3 HC-SR04 ultrasonic sensors to use just interrupts to create a scheduler to get readings from the ultrasonic sensors in a real time scenario.

Equipment & Requirements (for current configuration): 
- Arduino Uno R3
- Some type of Computer
- Arduino IDE [Download here](https://www.arduino.cc/en/software/)
- 3 HC-SR04 Ultrasonic Sensors
- Breadboard
- USB Type-B cable
- 3 560 ohm resistors
- 3 220 ohm resistors
- 3 leds
- External 5V power source (untested but recommended)
              
For initial set up go to [Setup](https://github.com/bdadams03/Real-Time-Systems-Final-Project/tree/main/Setup) where you will find information for setting up the hardware and code for the project.

The report I did in class for the projects can be found at [report](https://github.com/bdadams03/Real-Time-Systems-Final-Project/tree/main/report). This contains the midterm report I created as a step in the process of creating my final report which can also be found. There is also a video presentation for the project.
